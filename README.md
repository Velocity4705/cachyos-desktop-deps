# Desktop Environment Dependencies
This Script (de_manager.sh) provides the dependencies for all the Desktop Environments from **CachyOS**. Note that everything here is **Arch**.
## To install the script:
```
git clone https://github.com/Velocity4705/cachyos-desktop-deps.git
cd cachyos-desktop-deps
chmod +x de_manager.sh
```

## Run the file using:
```
./de_manager.sh
```
